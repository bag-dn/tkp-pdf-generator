# Генератор ТКП

[👉 Открыть](https://bag-dn.github.io/tkp-pdf-generator/)

Браузерный генератор технико-коммерческих предложений (ПТИ/КП) в PDF.

Принимает таблицы тех. характеристик и спецификаций (`.xlsx`/`.xls`) и схему
(`.pdf`/`.png`/`.jpg`/`.svg`) — собирает из них альбомный PDF формата А3 с рамкой,
штампами ЕСКД, ГОСТ-шрифтом и выделяемым текстом.

Вся генерация происходит на клиенте, бэкенд не нужен.

## Запуск

```bash
npm i
npm run dev
```

## Шрифт

OpenGost Type A (Nikita Volchenkov), ГОСТ 2.304-81.
Лицензия SIL OFL 1.1 — `src/assets/fonts/OFL.txt`.
