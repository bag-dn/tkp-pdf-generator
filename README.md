# ТКП generator

Клиентский генератор технико-коммерческого предложения (ПТИ/КП) в PDF.

Загружаются таблицы тех. характеристик и спецификаций (`.xlsx`/`.xls`) и,
опционально, схема (`.pdf`/`.png`/`.jpg`/`.svg`). На выходе — A3-альбомный PDF
с выделяемым текстом, штампами ЕСКД, рамкой и встроенным чертёжным шрифтом.

Всё работает в браузере: данные не покидают компьютер, сервер не нужен.

## Разработка

```
npm install
npm run dev       # http://localhost:5173
npm run build     # сборка в dist/
npm run preview   # предпросмотр сборки
```

## Деплой

GitHub Pages, автоматически при пуше в `main` (`.github/workflows/deploy.yml`).
`base` в `vite.config.js` должен совпадать с именем репозитория.

## Шрифт

OpenGost Type A (Nikita Volchenkov) — SIL OFL 1.1, ГОСТ 2.304-81.
Текст лицензии: `src/assets/fonts/OFL.txt`.
